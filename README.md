# Brain Tumor Segmentation and Classification using 3D U-Net

## Overview
This project performs brain tumor segmentation and classification from MRI images using a hybrid deep learning approach.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

## Features
- MRI image preprocessing
- 3D U-Net based tumor segmentation
- CNN/VGG based tumor classification
- Performance evaluation using Accuracy, Precision, Recall and F1-Score

## Result
The model achieved approximately 88% classification accuracy.
